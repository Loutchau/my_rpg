<h1 align="center">My RPG<br/>Epitech projet</h1>
<p>
</p>

Make a real and fonctional RPG game in C and CSFML library

## Install

```sh
git clone git@github.com:Loutchau/my_rpg.git
```

## Usage

```sh
make
```

After the loading enter : 

```sh
./my_rpg
```
